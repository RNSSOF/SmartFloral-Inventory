# 🌸 SmartFloral Inventory

نظام إدارة مخزون ذكي للورد باستخدام الذكاء الاصطناعي للتنبؤ والتسعير الديناميكي.

## 📦 الميزات الرئيسية

- **إدارة المخزون التنبؤية**: تسجيل الأزهار وحساب تلقائي لتاريخ التلف
- **التنبيهات الذكية**: تنبيهات تلقائية للأزهار القريبة من التلف
- **التسعير الديناميكي**: خصومات تلقائية بناءً على عمر الوردة
- **لوحة تحكم شاملة**: عرض إحصائيات المخزون والمبيعات والإيرادات
- **تتبع المبيعات**: سجل كامل للمبيعات والأرباح

## 🛠️ المتطلبات

- Node.js (v14+)
- npm أو yarn

## 🚀 البدء السريع

### 1️⃣ تثبيت المشروع

```bash
# كلون المستودع أو انسخ المجلد

# انتقل إلى المجلد
cd SmartFloral-Inventory

# ثبت المكتبات الأساسية
npm install
```

### 2️⃣ إعداد قاعدة البيانات

```bash
# ادخل إلى مجلد backend
cd backend

# ثبت المكتبات
npm install

# يمكنك تشغيل البرنامج لإنشاء قاعدة البيانات تلقائياً
# node db.js
```

### 3️⃣ تشغيل القطاع الخلفي (Backend)

```bash
# من داخل مجلد backend
npm run dev
```

الخادم سيعمل على: `http://localhost:5000`

### 4️⃣ تشغيل القطاع الأمامي (Frontend)

```bash
# في نافذة terminal جديدة
cd frontend

# ثبت المكتبات
npm install

# شغل التطبيق
npm start
```

التطبيق سيفتح تلقائياً على: `http://localhost:3000`

## 🔌 نقاط النهاية (API Endpoints)

### الأزهار
- `GET /api/flowers` - الحصول على جميع الأزهار
- `POST /api/flowers` - إضافة وردة جديدة
- `GET /api/flowers/:id` - الحصول على وردة محددة
- `PUT /api/flowers/:id/quantity` - تحديث كمية الوردة
- `DELETE /api/flowers/:id` - حذف وردة

### التنبيهات
- `GET /api/alerts` - الحصول على جميع التنبيهات
- `GET /api/alerts/critical` - الحصول على التنبيهات الحرجة
- `POST /api/alerts` - إنشاء تنبيه جديد
- `PUT /api/alerts/:id/resolve` - حل التنبيه

### التسعير
- `GET /api/pricing` - الحصول على سياسات التسعير
- `POST /api/pricing/apply-discount` - تطبيق خصم
- `GET /api/pricing/stats/summary` - إحصائيات التسعير

### المبيعات
- `POST /api/sales` - تسجيل مبيعة جديدة
- `GET /api/sales` - الحصول على سجل المبيعات
- `GET /api/sales/stats/daily` - إحصائيات يومية

### لوحة التحكم
- `GET /api/dashboard/summary` - ملخص البيانات
- `GET /api/dashboard/critical-flowers` - الأزهار الحرجة
- `GET /api/dashboard/performance` - الأداء العامة
- `GET /api/dashboard/top-alerts` - أهم التنبيهات
